同志Bot
— Discordサーバー向け社会主義語録ボット
機能 導入 GitHub サーバーに招待

故きを温ねて新しきを知る

キーワード検出に応じて社会主義語録を自動投稿するDiscordボットです。

今すぐ招待 導入手順

主な機能

キーワード検出
メッセージからキーワードを検出し語録を表示
スラッシュコマンド
/random_quoteでいつでも語録を呼び出し
安全設計
Bot自身の投稿は削除可能、誤爆防止

クイックスタート


# 環境準備
python -m venv venv
source venv/bin/activate
pip install -r requirements.txt

# .env に DISCORD_TOKEN を設定
python bot.py
